## Frequently Asked Questions
### Q: What goals should a real estate agent set??
**Key goals: annual GCI (gross commission income), number of transactions (listings and sales separately), leads generated per month, conversion rates, average commission per transaction, and client satisfaction scores. Set specific numbers, not vague intentions.**

### Q: How do you track real estate goals effectively??
**Review weekly: how many leads did you generate, how many showings, how many offers, how many closings. Track everything in one place. If you're behind pace after 30 days, adjust your strategy — don't just keep going the same way.**

### Q: What's a realistic first-year agent goal??
**Varies by market, but a reasonable first-year target is 6-10 transactions. Focus on learning the process, building your database, and getting great client reviews. Year 2 and 3 will compound significantly if you build correctly in year 1.**
