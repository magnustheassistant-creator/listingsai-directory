# Real Estate Showing Feedback Form — ListingsAI

**AI-generated showing feedback forms for real estate agents to collect buyer impressions after every showing.** Works in any browser — no signup required.

## What This Tool Does

Generates professional showing feedback forms that help agents collect actionable buyer feedback after every showing. Use the feedback to improve your listings, price correctly, and close more deals.

## How to Use

1. Enter the **property address, agent name, and listing price**
2. Select a **form type** (Buyer Feedback, Seller Summary, Quick Card, or Detailed)
3. Choose which **sections to include** (rating, price perception, likes, concerns, offer likelihood)
4. Click **Generate Feedback Form**
5. Copy, download, or email the form directly to buyers

## Key Features

- 4 different form types for different use cases
- Customizable sections (rating, price perception, concerns, offer likelihood)
- Designed for text/email distribution (high response rates)
- Seller summary report compiles all feedback into one view
- Works alongside CRM and marketing automation tools

## Use Cases

- Collecting buyer feedback after every showing
- Compiling weekly feedback summaries for sellers
- Identifying price and condition issues early
- Training agents on the importance of feedback collection
- Building data to support price adjustments
