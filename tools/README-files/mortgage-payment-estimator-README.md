# Mortgage Payment Estimator

Estimate monthly mortgage payments instantly. Enter home price, down payment, rate, and term to get PITI breakdown with PMI estimate.

## Features

- PITI calculation (Principal, Interest, Taxes, Insurance)
- PMI estimate for down payments under 20%
- Multiple term options (15, 20, 30 year)
- Monthly vs biweekly payment comparison
- Total interest paid over life of loan
