# CMA Report Generator

Free comparative market analysis tool for real estate agents. Generate professional CMA reports in under 5 minutes with comps, pricing analysis, and fair market value estimates.

## Features

- Complete CMA report generation from property details
- Fair market value range (low/mid/high) with supporting comps
- Pricing strategy recommendations based on market conditions
- Price-per-square-foot calculation for every property
- Market condition indicator (seller's/neutral/buyer's market)
- Copy to clipboard and print functionality
- Mobile responsive — works on any device
